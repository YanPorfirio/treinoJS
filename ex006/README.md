## Verificador de Horas

---

### Descrição

Este projeto é um verificador de horas que exibe a hora atual do sistema e uma imagem correspondente ao período do dia (manhã, tarde ou noite).

### Funcionalidades

- Exibe a hora atual do sistema
- Exibe uma imagem correspondente ao período do dia
- Exibe uma mensagem correspondente ao período do dia

### Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

### Como Executar

1. Clone o repositório
2. Abra o arquivo `ex006.html` no navegador

ou

1. Vá até o repositório no Github
2. Clique em "View Live" no canto superior direito
